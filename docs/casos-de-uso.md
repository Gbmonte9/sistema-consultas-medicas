# ⚙️ Casos de Uso do Sistema

| Código | Nome | Atores | Descrição |
|---------|------|---------|-----------|
| C1 | Cadastrar paciente | Admin | Permite criar o perfil de um paciente no sistema |
| C2 | Cadastrar médico | Admin | Registra médicos com CRM e especialidade |
| C3 | Agendar consulta | Paciente | Permite o agendamento de uma consulta médica |
| C4 | Cancelar consulta | Paciente / Médico | Cancela uma consulta já agendada |
| C5 | Visualizar consultas | Médico / Paciente | Exibe a agenda de consultas |
| C6 | Registrar histórico | Médico | Registra informações sobre a consulta e receita |
| C7 | Gerar relatório | Admin | Gera relatório em PDF de consultas por período |
